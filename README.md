### 팀의 방향성

1. 자주 빠르게 소통하자
2. 6시 퇴근을 추구하자
3. issues를 적극 활용하자



### 브랜치

파트 별 `dev` 브랜치에서 분기

```javascript
team-06 (default)
    |
    |--> dev/BE
    |      |
    |      |--> feature...
    |
    |--> dev/iOS
    |      |
    |      |--> feature...
```

- BE 진행 상황 - [dev/BE](https://github.com/ehdrhelr/baseball/tree/dev/BE)
- iOS 진행 상황 - [dev/iOS](https://github.com/ehdrhelr/baseball/tree/dev/iOS)
